📂 Dataset Reference
All columns in the Titanic dataset — raw inputs and engineered features
🗂️ Raw Dataset Columns
ColumnTypeDescription
PassengerIdIntegerUnique ID — not used in model
SurvivedBinaryTarget — 0 = Died, 1 = Survived
PclassIntegerTicket class — 1 = First, 2 = Second, 3 = Third
NameStringFull name — used to extract Title
SexStringPassenger gender — male or female
AgeFloatAge in years — ~20% missing, filled by median per Pclass+Sex
SibSpIntegerSiblings or spouses aboard
ParchIntegerParents or children aboard
TicketStringTicket number — dropped, too many unique values
FareFloatTicket price in British pounds (£)
CabinStringCabin number — ~77% missing, dropped
EmbarkedStringPort — S=Southampton, C=Cherbourg, Q=Queenstown
⚙️ Engineered Features
FeatureFormulaWhy It Helps
FamilySizeSibSp + Parch + 1Small families had higher survival
IsAlone1 if FamilySize == 1Solo travellers had lower survival odds
TitleExtracted from NameCaptures social status — Mr, Mrs, Miss, Master, Rare
